# coolidgeadvisory.com

Corporate website for Coolidge Advisory, Inc.

**Live:** [coolidgeadvisory.com](https://coolidgeadvisory.com)

## Stack

- Static HTML / CSS / vanilla JS
- Google Fonts (Libre Baskerville + DM Sans)
- Hosted on GitHub Pages
- Domain via Namecheap
- SSL via GitHub Pages (automatic)

## Deployment

Pushes to `main` branch auto-deploy via GitHub Pages.

## Files

```
index.html          Main site
favicon-32.png      Browser favicon
coolidge-icon-512.png   Social sharing / OG image
CNAME               Custom domain config
robots.txt          Search engine directives
.nojekyll           Bypass Jekyll processing
```
